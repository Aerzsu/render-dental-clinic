How to get the Google Maps embed URL

Follow these steps:

  1. 1 Visit Google Maps
  2. 2 Search for your clinic location
  3. 3 Click the "Share" button
  4. 4 Select the "Embed a map" tab
  5. 5 Click "COPY HTML"
  6. 6 Paste the code here and find src="..."
  7. 7 Copy only the URL inside the quotes (starts with https://www.google.com/maps/embed...)

Important: Copy only the URL, not the entire <iframe> HTML code

How Auto-Approval Works

  • Automatically approves appointments from patients with completed appointment history (if "Require Existing Patients" is enabled)
  • Requires manual review for first-time patients to maintain quality control
  • Always blocks same-day bookings (appointments must be at least 24 hours in advance)
  • Prevents double-booking â€” patients cannot book multiple appointments on the same day
  • Sends confirmation email immediately for auto-approved appointments

Tip: Start with "Require Existing Patients" enabled to ensure first-time patients receive personal attention, then adjust based on your workflow preferences.
